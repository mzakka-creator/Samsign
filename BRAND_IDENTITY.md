# SAM SIGN - Brand Identity Guide

## Brand Overview

**Company Name:** SamSign (stylized as **SAM**)  
**Tagline:** "Art & Experience.. Driven by Performance."  
**Industry:** Professional Advertising Solutions & Signage  
**Established:** Since 2018  
**Expertise:** Indoor signs, outdoor signs, and comprehensive signage solutions

---

## Visual Identity

### Logo Style
- **Bold, geometric, modern typography**
- The letter "A" forms a distinctive triangle shape
- Clean, professional aesthetic
- Strong visual impact suitable for industrial/construction contexts

### Primary Color Palette

#### Yellow/Gold (#F4C430)
- **Purpose:** Energy, visibility, construction/industry
- **Usage:** Primary brand color, CTAs, highlights, key messaging
- **RGB:** 244, 196, 48
- **CMYK:** 0, 20, 80, 4
- **Psychology:** Attention-grabbing, professional energy, industry authority

#### Black (#000000)
- **Purpose:** Professionalism, authority, power
- **Usage:** Primary text, logo, headers, strong contrast elements
- **RGB:** 0, 0, 0
- **CMYK:** 0, 0, 0, 100
- **Psychology:** Strength, reliability, professional credibility

#### White (#FFFFFF)
- **Purpose:** Clarity, precision, cleanliness
- **Usage:** Backgrounds, negative space, text on dark backgrounds
- **RGB:** 255, 255, 255
- **CMYK:** 0, 0, 0, 0
- **Psychology:** Clean, modern, professional clarity

#### Grey (Various Shades)
- **Purpose:** Balance, sophistication, industrial strength
- **Usage:** Secondary text, borders, subtle backgrounds, depth
- **Recommended Shades:**
  - Light Grey: #F5F5F5 (backgrounds)
  - Medium Grey: #808080 (secondary text)
  - Dark Grey: #333333 (subtle accents)
- **Psychology:** Stability, balance, industrial reliability

## Typography

### Primary Fonts
- **Headings:** Bold, geometric sans-serif (e.g., Montserrat Bold, Inter Bold, or similar)
- **Body:** Clean, modern sans-serif (e.g., Inter, Roboto, or system fonts)
- **Display:** Large, impactful fonts for hero sections

### Typography Hierarchy
- **H1:** Large, bold, confident (48-72px)
- **H2:** Strong subheadings (32-48px)
- **H3:** Section headers (24-32px)
- **Body:** Readable, clean (16-18px)
- **Small:** Supporting text (14px)

### Typography Principles
- Generous letter-spacing for headers
- Clear hierarchy with size and weight
- High contrast for readability
- Bold but not overwhelming

## Design Principles

### Core Philosophy
1. **Bold but not loud** - Strong presence without being overwhelming
2. **Simple but not basic** - Clean design with sophisticated details
3. **Professional but not corporate** - Approachable yet authoritative
4. **Modern but not trendy** - Timeless design that won't date quickly

Show leadership through:

## Application Guidelines

### Color Usage
- **Yellow/Gold:** Use sparingly for maximum impact (CTAs, highlights, key elements)
- **Black:** Primary text and strong elements
- **White:** Clean backgrounds and negative space
- **Grey:** Supporting elements, subtle depth

### Logo Usage
- Always maintain clear space around logo
- Use on contrasting backgrounds
- Never distort or alter proportions
- Minimum size for legibility

### Imagery Style
- Professional, high-quality photography
- Show real completed projects
- Industrial/construction contexts
- Clean, well-lit compositions
- Focus on craftsmanship and detail

### Spacing & Layout
- Generous margins and padding
- Consistent grid system
- Clear visual hierarchy
- Balanced composition
